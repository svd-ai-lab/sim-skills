# Standalone Python Interpreter

A standalone Python interpreter is bundled in the distribution packages. It is the same interpeter embedded in BETA applications. It uses the Python environment of the BETA distribution but has not ties to BETA Application, functionality or libraries. It can run as vanilla interpreter. This Python 3.11.9 interpreter can be launched from the following wrapper scripts:
