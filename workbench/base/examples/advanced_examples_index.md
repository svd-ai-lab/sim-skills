# PyWorkbench Advanced Examples

A collection of examples showcasing complex workflows generated with PyWorkbench and other PyAnsys libraries.

## Examples

- [Cooled Turbine Blade](cooled_turbine_blade.md) - Stress analysis of a cooled turbine blade using PyMechanical integration
- [Cyclic Symmetry Analysis](cyclic_symmetry_analysis.md) - Harmonic response analysis with cyclic symmetry using PyMechanical
- [Axisymmetric Rotor](axisymmetric_rotor.md) - 2D axisymmetric and 3D rotor modal analysis using PyMechanical
- [Material Designer Workflow](material_designer_workflow.md) - Parameterized material property analysis using Material Designer
